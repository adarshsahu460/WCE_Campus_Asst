# Machine Learning Syllabus
## Third Year Computer Engineering - Semester II

### Course Code: CSE301
### Credits: 4 (3-0-2)
### Total Hours: 45 Lecture + 30 Practical

---

## Course Objectives
1. Understand fundamental concepts of machine learning
2. Learn various supervised and unsupervised learning algorithms
3. Implement ML algorithms using Python libraries
4. Apply ML techniques to real-world problems

---

## Unit 1: Introduction to Machine Learning (8 Hours)

### 1.1 Fundamentals
- Definition and types of machine learning
- Supervised vs Unsupervised vs Reinforcement Learning
- Applications of machine learning
- Machine learning workflow

### 1.2 Mathematical Foundations
- Linear algebra basics (vectors, matrices)
- Probability and statistics review
- Optimization basics (gradient descent)

### 1.3 Data Preprocessing
- Feature scaling and normalization
- Handling missing data
- Feature encoding (one-hot, label encoding)
- Train-test split, cross-validation

---

## Unit 2: Supervised Learning - Regression (9 Hours)

### 2.1 Linear Regression
- Simple linear regression
- Multiple linear regression
- Polynomial regression
- Regularization (L1, L2, Elastic Net)

### 2.2 Regression Metrics
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)
- R-squared score

### 2.3 Advanced Topics
- Feature selection techniques
- Bias-variance tradeoff
- Overfitting and underfitting

---

## Unit 3: Supervised Learning - Classification (10 Hours)

### 3.1 Logistic Regression
- Binary classification
- Multi-class classification
- Softmax regression

### 3.2 Decision Trees
- Information gain and entropy
- Gini impurity
- Tree pruning techniques

### 3.3 Ensemble Methods
- Random Forests
- Gradient Boosting (XGBoost, LightGBM)
- AdaBoost
- Bagging vs Boosting

### 3.4 Support Vector Machines
- Linear SVM
- Kernel SVM (RBF, Polynomial)
- Support vectors and margins

### 3.5 Classification Metrics
- Confusion matrix
- Precision, Recall, F1-score
- ROC-AUC curve
- Multi-class classification metrics

---

## Unit 4: Unsupervised Learning (9 Hours)

### 4.1 Clustering
- K-Means clustering
- Hierarchical clustering
- DBSCAN
- Cluster evaluation metrics (Silhouette score)

### 4.2 Dimensionality Reduction
- Principal Component Analysis (PCA)
- t-SNE
- Feature importance

### 4.3 Association Rule Mining
- Apriori algorithm
- FP-Growth
- Support, confidence, lift

---

## Unit 5: Neural Networks and Deep Learning Basics (9 Hours)

### 5.1 Artificial Neural Networks
- Perceptron model
- Multi-layer perceptron (MLP)
- Activation functions (ReLU, Sigmoid, Tanh)
- Backpropagation

### 5.2 Training Neural Networks
- Loss functions
- Optimizers (SGD, Adam, RMSprop)
- Batch normalization
- Dropout regularization

### 5.3 Introduction to Deep Learning
- Convolutional Neural Networks (CNN) overview
- Recurrent Neural Networks (RNN) overview
- Transfer learning basics

---

## Practical List (30 Hours)

1. Data preprocessing and visualization using Pandas, Matplotlib
2. Linear regression implementation and evaluation
3. Logistic regression for binary classification
4. Decision tree and random forest implementation
5. SVM classification with different kernels
6. K-Means clustering on real datasets
7. PCA for dimensionality reduction
8. Neural network implementation using TensorFlow/Keras
9. Mini project: End-to-end ML pipeline

## Course Outcomes

After completing this course, students will be able to:
1. Explain fundamental concepts of machine learning
2. Implement and evaluate supervised learning algorithms
3. Apply unsupervised learning techniques for clustering and dimensionality reduction
4. Design and train basic neural networks
5. Develop end-to-end ML solutions for real-world problems
